# 👨‍💻 About This Repository

This repository contains carefully selected coding problems designed to improve:

- Logic-building skills
- Java programming skills
- Data Structures & Algorithms understanding
- Coding interview preparation

The problems are organized from **Beginner → Advanced** so developers can improve their problem-solving skills step by step.

# 🟢 Level 1 — Basic Logic

Focus: loops, conditions, and math logic.

Problems include:

- Reverse a number
- Check a palindrome number
- Count digits in a number
- Fibonacci series
- Prime number
- Factorial
- GCD
- LCM
- Multiplication table
- Armstrong number

These problems help build **core logical thinking skills**.

---

# 🟢 Level 2 — Array Basics

Focus: loops and array manipulation.

Examples include:

- Find the largest element in the array
- Find the second largest element
- Reverse an array
- Sum of array elements
- Count even and odd numbers
- Remove duplicates
- Find the missing number
- Rotate array

---

# 🟢 Level 3 — String Problems

Focus: string manipulation and logic.

Examples include:

- Reverse string
- Palindrome string
- Count vowels and consonants
- Character frequency
- Remove spaces from the string
- Count words
- First non-repeating character
- Check anagram strings

---

# 🟡 Level 4 — Intermediate Coding Problems

Commonly asked in coding interviews.

Examples include:

- Two Sum
- Move zeros to end
- Intersection of arrays
- Union of arrays
- Kadane’s Algorithm
- Stock buy and sell problem
- Majority element

---

# 🟠 Level 5 — Advanced Interview Problems

Examples include:

- Longest substring without repeating characters
- Group anagrams
- Top K frequent elements
- Kth largest element
- Merge overlapping intervals
- LRU Cache implementation

# 🔴 Level 6 — Big Tech Interview Problems / MAANG Level Problems
